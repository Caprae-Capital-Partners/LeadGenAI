Getting Started

Welcome to the Lead Management System. This guide will help you get started with managing your leads effectively.

Quick Start

To begin managing your leads:

  1. Log in to your account
  2. Navigate to "Edit & Export Leads" to view your leads
  3. Use "Single Import" or "Bulk Import" to add new leads
  4. Export your data as needed

Importing Leads

You can import leads in two ways:

Single Import

Use the single import form to add individual leads:

  1. Click on "Single Import" in the sidebar
  2. Fill in the lead's information
  3. Click "Submit" to add the lead

Bulk Import

To import multiple leads at once:

  1. Click on "Bulk Import" in the sidebar
  2. Prepare your CSV file with the required columns
  3. Upload the file and map the columns
  4. Review and confirm the import

Managing Leads

LeadGen provides powerful search and filtering capabilities to help you find specific leads quickly.

Basic Search

Use the search bar at the top of the Leads page to perform a quick search across all fields. This is useful for finding leads by name, email, company, or any other information.

Filtering

For more targeted searching, use the filtering options:

Basic Filters

  • Company - Filter by company name
  • Location - Filter by city or state
  • Role/Title - Filter by job title or position
  • Lead Score - Filter by assigned lead score

Advanced Filtering

Click the "Advanced Filters" button to access more powerful filtering options:

  • Industry - Filter by industry sector
  • Business Type - Filter by business model (B2B, B2C, etc.)
  • State - Filter by specific state
  • Employee Size - Filter by company size
  • Revenue Range - Filter by company revenue
Pro Tip
Combine multiple filters to narrow down your results even further. For example, search for leads in a specific industry AND with a particular employee size.

Exporting Data

LeadGen allows you to export your lead data for use in other systems or for reporting purposes.

Exporting Selected Leads

1
Select Leads
Use the checkboxes on the left side of the leads table to select the leads you want to export. You can use the "Select All" checkbox at the top to select all leads that match your current search/filter criteria.
2
Choose Export Format
Select your preferred export format (CSV or Excel) from the dropdown menu.
3
Export Data
Click the "Export Selected" button to generate and download your export file.

Export Options

The system offers two export formats:

  • CSV (Comma Separated Values) - A universal format that can be opened in any spreadsheet application or imported into other systems.
  • Excel (.xlsx) - A native Microsoft Excel format that preserves formatting and is ideal for further analysis in Excel.
Pro Tip
Use filters before exporting to create targeted export lists for specific campaigns or follow-up actions.

User Management

You can update lead information as you gather more data or need to make corrections.

Updating Lead Status

All users can update the lead status:

1
Find the Lead
Use the search and filter functions to locate the lead you want to update.
2
Change Status
Find the status dropdown in the Actions column for that lead and select the new status (New, Contacted, Qualified, Proposal, Closed).
3
Save Changes
The change is automatically saved when you select a new status.

Editing Lead Details

Only Admin and Developer roles can edit full lead details:

1
Find the Lead
Use the search and filter functions to locate the lead you want to edit.
2
Access Edit Mode
Click the "Edit" button in the Actions column for that lead.
3
Make Changes
Update the lead information in the edit form. You can modify fields like name, contact information, company details, and notes.
4
Save Changes
Click the "Save" button to commit your changes to the database.
Pro Tip
Keep your lead status updated regularly to maintain an accurate pipeline view and ensure proper follow-up actions.
